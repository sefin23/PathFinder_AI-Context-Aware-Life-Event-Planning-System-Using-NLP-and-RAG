# 🗺️ Pathfinder AI — Brand Identity Guide

This guide defines the visual language of the Pathfinder AI ecosystem. It serves as the "Source of Truth" for all branding and UI tokens.

> [!Note]
> For detailed UI components, animations, and interaction logic, see the [**Design System Spec**](./DESIGN_SYSTEM.md).

## 📐 Typography
We use a high-contrast pairing of Serif and Sans to create an "Editorial Intelligence" feel.

*   **Display / Hero**: `Playfair Display` — Warm, human, and premium. Used for page titles and milestones.
*   **System / Body**: `DM Sans` — Modern, clean, and highly legible for complex workflows.
*   **Technical / UI**: `JetBrains Mono` — Used for status markers, IDs, and technical precision.

## 💠 Design Tokens (UI Principles)
*   **Corners**: All interactive elements use "Soft" rounding:
    *   `--r-sm`: 8px (Inner cards)
    *   `--r-md`: 16px (Main cards)
    *   `--r-lg`: 24px (Large panels)
    *   `--r-pill`: 9999px (Buttons/Pills)
*   **Space**: Consistent 8px grid system (`--space-1` through `--space-5`).
